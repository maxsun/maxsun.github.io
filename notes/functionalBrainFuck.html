Functional Brainf-ck

Brainfuck is a programming language comprised of only 8 symbols. Despite the simplicity of the langauge syntax, Brainfuck is actually a Turing complete programming language. The cool consequence of this, is that Brainfuck is capable of computing all the same algorithms your favorite language can compute. Essentially, Brainfuck simulates a Turing machine with these 8 commands:

  1. > Move pointer right.
  2. < Move pointer left.
  3. + Increase the byte at the pointer.
  4. - Decrease the byte at the pointer.
  5. . Output the byte at the pointer.
  6. , Wait for one byte of input, stored at the pointer.
  7. [ If the current byte is 0, skip to matching "]".
  8. ] If the current byte is not 0, move back to the matching "["

If you're familiar with the variety of features packed into most popular programming languages, you might be wondering: "where are my variables?" or "what about functions?". It turns out, this is all you need to be able to solve the same problems you can solve with a fancy language like Python, it just might be a lot more confusing in Brainfuck. For example, this is the code to print "Hello World": 

+[-[<<[+[--->]-[<<<]]]>>>-]>-.---.>..>.<<<<-.<+.>>>>>.>.<<.<-.
