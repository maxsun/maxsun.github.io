Functional Brainf-ck

Brainfuck is an esoteric programming language comprised of only 8 symbols. Despite the simplicity of the langauge syntax, Bf is actually a Turing complete programming language. The cool consequence of this, is that Bf is capable of computing all the same algorithms your favorite language can compute. Essentially, Brainfuck simulates a Turing machine with these commands:

  1. > Move pointer right.
  2. < Move pointer left.
  3. + Increase the byte at the pointer.
  4. - Decrease the byte at the pointer.
  5. . Output the byte at the pointer.
  6. , Wait for one byte of input, stored at the pointer.
  7. [ If the current byte is 0, skip to matching "]".
  8. ] If the current byte is not 0, move back to the matching "["


Lambda Calculus is a formal system designed by Alonzo Church that also allows for computation of all algorithms also computable by a Turing machine. Similar to Bf, Lambda Calculus is extremely simple- consisting of only 3 syntactic rules.
